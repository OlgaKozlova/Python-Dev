## Задача 1 — порядок выполнения

Что будет выведено?

```python
x = 4
while x > 0:
    if x == 2:
        break
    x -= 1
    print(x)
else:
    print('done')
```

### Ответ

Вывод:

```
3
```

Пояснение:

- `x` уменьшается **после проверки `x == 2`**
- при `x == 2` выполняется `break`
- `else` не выполняется из-за `break`

## Задача 2 — continue и состояние

Что будет выведено?

```python
x = 0
while x < 5:
    x += 1
    if x % 2 == 0:
        continue
    print(x)
```

### Ответ

Вывод:

```
1
3
5
```

Пояснение:

- `continue` пропускает чётные значения
- состояние (`x += 1`) изменяется до `continue`

---

## Задача 3 — while … else

Будет ли выполнен `else`?

```python
x = 0
while x < 3:
    x += 1
    if x == 2:
        break
else:
    print('end')
```

### Ответ

`else` **не выполнится**.

Пояснение:

- цикл завершён через `break`
- `else` у цикла выполняется только при нормальном завершении

## Задача 4 — внимательность

Что будет выведено?

```python
x = 3
while x:
    print(x)
    x -= 1
```

### Ответ

Вывод:

```
3
2
1
```

Пояснение:

- `while x:` использует truthy / falsy
- `0` — falsy, цикл завершается
