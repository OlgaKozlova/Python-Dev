# День 11 — Практика

## Задача

**Что будет выведено и почему?**

```python
def log(msg):
    print(msg)
    return msg

x = None
result = x and log("A") or log("B")
print("result:", result)
```

---

## Решение

### Пошагово

1. `x` равен `None` → falsy
2. `x and log("A")`:
   - левый операнд falsy
   - `log("A")` **не вызывается**
   - результат всего выражения — `None`

3. `None or log("B")`:
   - левый операнд falsy
   - вызывается `log("B")`
   - печатается `B`
   - результат выражения — `"B"`

4. `result = "B"`
5. Печать результата

### Итоговый вывод

```
B
result: B
```

---

## Ключевая проверка понимания

- `and` и `or` возвращают значения
- функции могут **не вызваться вообще**
- порядок вычисления критичен

Если этот код читается без паузы — День 11 усвоен корректно.
