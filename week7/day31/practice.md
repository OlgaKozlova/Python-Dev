## Задача 1

Есть класс:

```python
class Counter:
    def __init__(self, start):
        value = start
```

1. Объясни, что здесь пойдёт не так.
2. Исправь код так, чтобы:
   - значение сохранялось в объекте
   - его можно было прочитать после создания экземпляра

## Задача 2

Напиши класс `User`, который:

- принимает `name` в `__init__`
- имеет метод `rename(new_name)`
- после вызова `rename` меняет имя у конкретного объекта

Пример использования:

```python
u = User("Olga")
u.rename("Maria")
print(u.name)  # Maria
```

---

## Проверка понимания (без кода)

Ответь письменно:

1. Почему `self` нельзя убрать из сигнатуры метода?
2. Что произойдёт, если внутри `__init__` написать `name = name`?
3. Чем метод отличается от функции на уровне модели Python?

---

## Решения (проверь себя)

### Задача 1 — решение

Ошибка: `value` — локальная переменная.

Правильно:

```python
class Counter:
    def __init__(self, start):
        self.value = start
```

---

### Ключевые ответы

1. `self` — способ передать ссылку на экземпляр; без него метод не знает, с каким объектом работает.
2. Переменная будет локальной и исчезнет после выполнения `__init__`.
3. Метод — это функция, связанная с объектом (bound method).
