## Задача 4 — Переназначение

```python
class B:
    n = 0
    def inc(self):
        self.n += 1

b1 = B()
b2 = B()
b1.inc()
b2.inc()

print(b1.n, b2.n, B.n)
```

Почему результат отличается от задачи 3?

---

# Решения

## Задача 1

Будет накопление, потому что default-список создаётся один раз и мутируется.

Исправление — использовать `None` и создавать новый список внутри функции.

---

## Задача 2

`acc = acc + [x]` создаёт новый список.
Default-объект не мутируется.

---

## Задача 3

`xs` — атрибут класса.
Метод мутирует общий список.
Все экземпляры видят одно и то же состояние.

---

## Задача 4

`int` immutable.
`self.n += 1` создаёт новый объект и записывает его в экземпляр.
Атрибут класса остаётся неизменным.
