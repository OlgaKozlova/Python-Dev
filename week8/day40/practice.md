## Задача 1

```python
a = [1, 2]
b = a
c = a[:]

a.append(3)

print(a)
print(b)
print(c)
print(a is b)
print(a is c)
print(b == c)
```

### Ответ

```
[1, 2, 3]
[1, 2, 3]
[1, 2]
True
False
False
```

**Почему:**

- `b` указывает на тот же объект.
- `c` — новый внешний список.
- `append` изменяет объект по месту.

## Задача 2

```python
a = [[1, 2], [3, 4]]
b = a[:]

a[0].append(99)

print(a)
print(b)
print(a is b)
print(a[0] is b[0])
```

### Ответ

```
[[1, 2, 99], [3, 4]]
[[1, 2, 99], [3, 4]]
False
True
```

**Почему:**

- Shallow copy.
- Внутренние списки общие.

## Задача 3 — copy vs deepcopy

```python
import copy

a = [[1, 2], [3, 4]]
b = copy.copy(a)
c = copy.deepcopy(a)

a[0].append(99)
a.append(["x"])
```

### Итоговое состояние

```
a: [[1, 2, 99], [3, 4], ['x']]
b: [[1, 2, 99], [3, 4]]
c: [[1, 2], [3, 4]]
```

### Ключевые проверки

```
a is b        → False
a == b        → False

a[0] is b[0]  → True
a[0] == b[0]  → True

a[0] is c[0]  → False
a[0] == c[0]  → False
```

### Ошибка доступа

```
b[2]
```

→ `IndexError`, потому что элемента с индексом 2 нет.
