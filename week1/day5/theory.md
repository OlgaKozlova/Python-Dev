# День 5 — Строки в Python

## 1. Строки как immutable-объекты

`str` в Python — **неизменяемый тип**.
Любой метод строки **не меняет объект**, а возвращает **новую строку**.

```python
s = "hello"
s.upper()
print(s)  # 'hello'
```

Это фундамент: если результат метода не присвоен — он потерян.

---

## 2. Методы изменения регистра

### `capitalize()`

Делает:

- первый символ — `upper`
- **все остальные** — `lower`

```python
"sAlutIk".capitalize()
# 'Salutik'
```

### `swapcase()`

Меняет регистр **каждого** символа на противоположный.

```python
"sAlutIk".swapcase()
# 'SaLUTiK'
```

### `title()`

Эвристически делает первую букву каждого "слова" заглавной.
Не понимает язык, работает по простым правилам.

```python
"they're fine".title()
# "They'Re Fine"
```

---

## 3. Удаление символов по краям

### `strip()`, `lstrip()`, `rstrip()`

Удаляют **только с краёв строки**.

```python
"  hello \n".strip()
# 'hello'
```

### С аргументом

Аргумент — **набор символов**, а не подстрока.

```python
"abcaba".strip("ab")
# 'c'
```

Алгоритм:

- слева: удалять, пока символ ∈ набору
- справа: то же
- середина **никогда не трогается**

```python
"abcaba".lstrip("ab")
# 'caba'

"abcaba".rstrip("ab")
# 'abc'
```

---

## 4. Поиск и замена

### `replace(old, new, count=None)`

Заменяет **подстроки** по всей строке.

```python
"banana".replace("a", "o")
# 'bonono'
```

`count` — максимальное число замен слева направо.

```python
"banana".replace("a", "o", 2)
# 'bonona'
```

Если `old` не найден — строка возвращается без изменений.

## 5. Разбиение и склейка

### `split()` без аргументов

Разбивает по **whitespace**:

- несколько подряд считаются одним
- пустые строки отбрасываются

```python
"  a   b\tc \n d  ".split()
# ['a', 'b', 'c', 'd']
```

### `split(sep)`

Использует **строго указанный разделитель**.
Пустые строки сохраняются.

```python
"a,,b,c,".split(",")
# ['a', '', 'b', 'c', '']
```

### `maxsplit`

Максимальное число **разбиений**, а не элементов.

```python
"a,b,c,d".split(",", 2)
# ['a', 'b', 'c,d']
```

### `join(iterable)`

Склеивает элементы итерируемого объекта.
Итерируется **по аргументу**, не по строке-приёмнику.

```python
",".join(["a", "b", "c"])
# 'a,b,c'
```

## 6. Проверочные методы

### `isalnum()`

True, если строка:

- не пустая
- состоит **только** из букв и/или цифр

```python
"abc123".isalnum()   # True
"abc 123".isalnum()  # False
"".isalnum()         # False
```

### `isspace()`

True, если строка:

- не пустая
- состоит **только** из whitespace-символов

```python
"   \t\n".isspace()  # True
" a ".isspace()      # False
```

## 7. Форматирование строк (f-strings)

### Базовый синтаксис

```python
name = "Bob"
age = 30

f"{name} is {age}"
```

Внутри `{}` — **выражение Python**, вычисляется **один раз**.

```python
x = 3
s = f"{x * 2}"
x = 100
print(s)  # 6
```

---

## 8. Формат-спецификаторы

Синтаксис:

```python
f"{value:spec}"
```

### Ширина и выравнивание

```python
f"{42:5}"    # '   42'
f"{42:<5}"   # '42   '
f"{42:^5}"   # ' 42  '
```

### Заполнение

```python
f"{42:0>5}"  # '00042'
```

### Числа с точкой

```python
x = 3.14159
f"{x:.2f}"   # '3.14'
f"{x:6.2f}"  # '  3.14'
```

Ширина — **минимальная**. Значения **никогда не обрезаются**.
