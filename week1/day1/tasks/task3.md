# Упражнение 3 — тело класса

## Что будет выведено и почему
```python
print("start")

class A:
    print("class body")
    def m(self):
        print("method")

print("end")

a = A()
```

## Ответ
```
start
class body
end
```

## Пояснение
- `print("start")` выполняется сразу  
- тело класса выполняется один раз при определении класса  
- метод `m` только создаётся, но не вызывается  
- `print("end")` выполняется  
- `A()` создаёт экземпляр, но не вызывает методы  

Тело класса — часть определения класса, а не логики программы.
