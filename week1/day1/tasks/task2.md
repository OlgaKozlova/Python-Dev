# Упражнение 2 — def не выполняется

## Что будет выведено и почему
```python
def f():
    print("inside f")

print("before")

def g():
    f()
    print("inside g")

print("after")
```

## Ответ
```
before
after
```

## Пояснение
- `def f()` и `def g()` создают функции, но не выполняют их тела  
- ни `f()`, ни `g()` не вызываются  
- выполняются только строки вне функций  

Определение функции ≠ выполнение функции.
