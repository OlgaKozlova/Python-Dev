# День 4. Практика по строкам

## Задача 1. Индексация

Дана строка:

```python
s = "abcdef"
```

1. Получить символ `'c'` двумя способами через индексацию.
2. Получить последний символ без использования `len()`.
3. Написать выражение, которое выбрасывает `IndexError`, и объяснить почему.

## Задача 2. Срезы

Дана строка:

```python
s = "0123456789"
```

Получить с помощью срезов:

1. Строку `"2345"`
2. Строку `"97531"`
3. Пустую строку (без условий и проверок длины)

---

## Задача 3. Неизменяемость и ссылки

Дан код:

```python
s = "abc"
t = s
s += "d"
```

Ответить:

1. Значение `s`
2. Значение `t`
3. Указывают ли `s` и `t` на один объект
4. На каком шаге создаётся новый объект и почему

---

## Решения и пояснения

### Задача 1

```python
s[2]
s[-4]
s[-1]
s[100]
```

Пояснение:

- допустимые индексы: `[-6 … 5]`
- `100` выходит за диапазон → `IndexError`

---

### Задача 2

```python
s[2:6]      # '2345'
s[::-2]     # '97531'
s[1:2:-1]   # ''
```

Пояснение:

- `s[::-2]` идёт с конца с шагом `-2`
- при отрицательном шаге старт должен быть правее стопа, иначе результат пустой

---

### Задача 3

Ответ:

1. `s == "abcd"`
2. `t == "abc"`
3. `s is t` → `False`
4. Новый объект создаётся при вычислении выражения `s + "d"`, так как строки immutable. После этого имя `s` перепривязывается к новому объекту.
