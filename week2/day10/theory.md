# День 10 — Кортежи и множества (tuple, set)

## Кортежи (`tuple`)

### Что это

`tuple` — упорядоченный **неизменяемый** контейнер. Это **value-type**, а не объект с поведением.

```python
point = (x, y)
key = (user_id, month, year)
```

### Зачем

- фиксированная форма данных
- сравнение по значению
- хешируемость (можно использовать как ключ `dict` и элемент `set`)
- декларация: «эти данные нельзя менять»

### Когда использовать

- 2–3 значения
- порядок очевиден
- нет поведения
- используется как ключ или состояние

### Когда НЕ использовать

- много полей и легко перепу
