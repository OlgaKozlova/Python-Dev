# День 7 — Ссылки и копирование (практика)

## Задача 1. Ссылки и поверхностное копирование

### Условие

```python
a = [1, 2, [3, 4]]
b = a
c = a.copy()

b[0] = 100
c[2].append(5)

print(a)
print(b)
print(c)
```

### Ответ

```python
[100, 2, [3, 4, 5]]
[100, 2, [3, 4, 5]]
[1, 2, [3, 4, 5]]
```

### Пояснение

- `a` и `b` — одна ссылка
- `c` — поверхностная копия
- вложенный список общий для всех

## Задача 2. Общие вложенные объекты

### Условие

```python
a = [1, 2]
b = [a, a]

c = b.copy()
c[0].append(3)

print(a)
print(b)
print(c)
print(b[0] is b[1], b[0] is c[0])
```

### Ответ

```python
[1, 2, 3]
[[1, 2, 3], [1, 2, 3]]
[[1, 2, 3], [1, 2, 3]]
True True
```

### Пояснение

- существует один вложенный список `a`
- `b` и `c` содержат ссылки на него
- поверхностное копирование не разрывает вложенные ссылки
