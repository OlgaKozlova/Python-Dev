# День 6 — Практика по спискам

## Задача 1. Пошаговое исполнение

### Условие

```python
lst = [1, 2, 3]

a = lst.append(4)
b = lst + [5]

lst.extend([6, 7])
c = lst[:]

lst.pop(0)
c.remove(3)
```

### Вопросы

1. Чему равны `a`, `b`, `c`?
2. Чему равен `lst` в конце?
3. Какие операции создают новый список?
4. Какие операции меняют существующий список?

---

### Решение

```python
a is None
b == [1, 2, 3, 4, 5]
lst == [2, 3, 4, 5, 6, 7]
c == [1, 2, 4, 5, 6, 7]
```

### Пояснение

- `append`, `extend`, `pop`, `remove` — меняют список
- `lst + [5]` — создаёт новый список
- `lst[:]` — создаёт копию
- `pop` — единственный метод, который **и меняет список, и возвращает значение**

## Задача 2. Ловушка с `+`

### Условие

```python
lst = [1, 2]
lst + [3, 4]
print(lst)
```

### Ответ

```python
[1, 2]
```

### Пояснение

Оператор `+` **никогда не меняет** существующий список. Он создаёт новый объект, который здесь никуда не сохранён.

## Задача 3. Изменяемость

### Условие

```python
lst = [[1, 2], [3, 4]]
lst[0].append(99)
```

### Ответ

```python
[[1, 2, 99], [3, 4]]
```

### Пояснение

Список хранит ссылки на вложенные объекты. Метод `append` меняет объект по ссылке.
