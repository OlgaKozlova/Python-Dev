# День 6 — Списки (list)

## 1. Что такое список

`list` — изменяемая (mutable), упорядоченная последовательность **ссылок на объекты**.

Ключевые свойства:

- список хранит **ссылки**, а не значения;
- список **изменяем**;
- элементы могут быть разных типов;
- поддерживает индексацию и срезы.

```python
lst = [1, 2, 3]
empty = []
chars = list("abc")  # ['a', 'b', 'c']
```

---

## 2. Список как контейнер ссылок

Список хранит ссылки на объекты. Перепривязка имени не влияет на содержимое списка.

```python
x = 10
lst = [x]
x = 20
# lst == [10]
```

Изменение вложенного объекта отражается в списке:

```python
lst = [[1, 2], [3, 4]]
lst[0].append(99)
# [[1, 2, 99], [3, 4]]
```

---

## 3. Индексация и присваивание

```python
lst = [10, 20, 30]
lst[0]      # 10
lst[-1]     # 30

lst[1] = 99
# [10, 99, 30]
```

В отличие от строк, присваивание по индексу допустимо, так как список изменяем.

---

## 4. Методы списков

### append(x)

Добавляет элемент в конец. Меняет список, возвращает `None`.

```python
lst.append(4)
```

### extend(iterable)

Добавляет элементы из итерируемого объекта.

```python
lst.extend([5, 6])
```

Отличие:

```python
lst.append([7, 8])   # [..., [7, 8]]
lst.extend([7, 8])   # [..., 7, 8]
```

### insert(i, x)

Вставляет элемент перед индексом `i`.

### remove(x)

Удаляет первое вхождение `x`. Если элемента нет — `ValueError`.

### pop([i])

Удаляет и возвращает элемент (по умолчанию последний).

```python
x = lst.pop()
```

### clear()

Очищает список, сохраняя объект.

```python
lst.clear()
```

### index(x), count(x)

- `index` возвращает индекс или выбрасывает `ValueError`
- `count` считает количество вхождений

---

## 5. sort и sorted

```python
lst.sort()          # меняет список
new = sorted(lst)  # создаёт новый список
```

---

## 6. Срезы списков

```python
lst = [1, 2, 3, 4]
a = lst[:]      # копия
```

Срез **всегда создаёт новый список**.

Но:

```python
lst[:] = [9, 9]
```

— меняет существующий объект списка.

---

## 7. Ментальная модель (обязательно)

- Методы списков **обычно меняют объект и возвращают None**
- Исключения: `pop`, `index`, `count`
- `+` и срезы создают **новые списки**
- `list`, `dict`, `set` передаются по ссылке

Это база для понимания копирования и побочных эффектов.
